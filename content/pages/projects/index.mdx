#### Bug Tracker

* A collaborative project management app. Users can create projects and mention bugs.
* It has a discussion panel where user can mention user and bug. 
* Also it has real-time notification system built using socket and Redis.
* Stack - TypeScript, ReactJS, Redux, NodeJS, Express, MongoDB, ChakraUI, Socket.io, Redis, Firebase

### Twitter Clone

* A Twitter-like web app implementing features like JSON Web Token for Authentication, posting a
tweet, like, retweet, follow a user, search, comments and nested replies.
* It uses Express server that listens and handles requests and fetches data from MySQL and Firebase.
* Stack - React, NodeJS, Express, MySQL, Firebase, ChakraUI
- | [Demo](https://www.youtube.com/watch?v=26lbkbUK7v4&ab_channel=AdityaPrabhat) | [Fronted](https://github.com/adityaprabhat01/twitter-frontend) | [Backennd](https://github.com/adityaprabhat01/twitter-frontend) |

### Notes App

* A note making app where users can create notebooks and notes. It uses React-Quill as an editor.
* It has auto save note feature which uses debouncing to reduce number of API calls.
* Stack - TypeScript, ReactJS, NodeJS, Express, MongoDB, GraphQL, Apollo.

- | [Demo](https://www.youtube.com/watch?v=V-N9udHM1wg&ab_channel=AdityaPrabhat) | [Fronted](https://github.com/adityaprabhat01/evernote-frontend) | [Backennd](https://github.com/adityaprabhat01/evernote-backend) | 

### Drive Clone

* A Google Drive-like app where users can upload, view, delete, download any type of file and create any directory structure.
* The uploaded files are stored in Firebase Storage and Directory tree is stored separately in Firestore.
* Stack - ReactJS, Redux, Firebase

- | [Demo](https://www.youtube.com/watch?v=ySC45Q32M6E&ab_channel=AdityaPrabhat) | [Live](https://distracted-curie-61fa72.netlify.app/) | [Code](https://github.com/adityaprabhat01/Drive) |
